**Key Technical Concepts**
-   **Secure Iframe-Safe Downloads:** A new core pattern was established. Since  cookies aren't accessible to JS, and  URLs are blocked, the solution is: (1) Frontend uses its valid session cookie to request a temporary, single-use token from a dedicated endpoint. (2) Frontend opens a new tab with the file URL, passing the temporary token in the query string. (3) Backend auth logic validates this token as a fallback.
-   **AI-Driven Data Entry & Validation:** Using a vision model (Claude Sonnet via ) to parse unstructured documents (DDTs) and pre-fill structured forms.
-   **Conditional Business Logic:** Implemented a key compliance workflow by linking digital records (certificates) to physical events (material arrivals documented by DDTs), preventing traceability records from being created without proof of delivery.
